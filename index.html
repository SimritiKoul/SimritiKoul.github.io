Simriti Koul


Hello! I'm Simriti Koul and I'm a Cyber enthusiast!

I received my Masters degree Information Security and Assurance with concentration in Applied Cyber Security from Department of Computer Science at George Mason University (GMU) in May 2023. I have spent time working at Blueicon Inc., North Carolina A&T State University, Resilience Inc. and George Mason University.

Throughout my career, I have amassed a diverse and robust experience in cybersecurity, software development, and auditing. As a Software Engineer at Blueicon Inc., I developed high-quality software applications with a strong emphasis on security, implementing secure coding practices and integrating security requirements into the software development lifecycle. My role involved threat modeling, risk assessments, code reviews, and security testing, ensuring adherence to security standards and staying current with emerging security trends. At North Carolina Agricultural and Technical State University, I conducted detailed information system audits, utilizing frameworks like NIST and ISO 27002, and performed multifaceted audits to safeguard institutional integrity. My tenure as a Mobile Application Developer at Resilience Inc. saw me developing and optimizing mobile apps, leading cross-functional teams, and enhancing backend infrastructure with AWS and VMware. As a Student Research Assistant at George Mason University, I secured NSF funding, developed an innovative eye-tracking program using Python and OpenCV, and utilized the Twitter API for data extraction and analysis. Additionally, during my internship at Netilly Digital LLP, I engineered an AI-powered Healthcare Chat-Bot, integrating deep learning and NLP technologies.

These roles have equipped me with a comprehensive skill set in cybersecurity, software engineering, and data analysis, making me well-prepared for advanced research and development in cybersecurity.

My interested topics include:

  • Cryptography

  • Secure Software Development

  • Cyber Attacks via Social Media
