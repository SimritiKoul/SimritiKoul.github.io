Simriti Koul


Hello! I'm Simriti Koul and I'm a Cyber enthusiast!

I received my Masters degree Information Security and Assurance with concentration in Applied Cyber Security from Department of Computer Science at George Mason University (GMU) in May 2023. I have spent time working at Blueicon Inc., North Carolina A&T State University, Resilience Inc. and George Mason University.

At Blueicon Inc. in Ashburn, VA, I have been pivotal in developing high-quality software applications with a strong emphasis on security from inception through deployment. My role involved utilizing programming languages such as Python, JavaScript, and C++ to create robust software solutions while implementing secure coding practices to prevent vulnerabilities like SQL injection, cross-site scripting (XSS), and buffer overflows. I integrated security requirements into the software development lifecycle (SDLC) and conducted threat modeling to identify and mitigate potential security risks. Additionally, I performed regular risk assessments, organized code review sessions, and developed and executed security test plans, including static and dynamic code analysis, penetration testing, and vulnerability assessments. Working closely with cross-functional teams, I ensured the secure design and implementation of software solutions. I also provided security guidance and training to development teams and stayed current with emerging security trends and technologies to continuously improve the security posture of our applications. My participation in security audits and implementation of recommendations further enhanced the security of our software products.
